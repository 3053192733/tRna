# tRna - 生物信息学工具集合

## 1. 产品概述

**tRna** 是一个专注于生物信息学的在线工具集合网站，帮助研究人员、学生和生物科技爱好者快速完成 DNA/RNA 序列分析、氨基酸查询等日常生物计算任务。

- **核心价值**：将复杂的生物信息学计算可视化、直观化，通过流畅的动画让抽象的分子生物学概念变得可触摸
- **目标用户**：生物学学生、研究人员、遗传工程爱好者、生物科技公司

## 2. 核心功能

### 2.1 功能模块

1. **首页**：工具导航、DNA 双螺旋动画背景、快速入口
2. **DNA/RNA 分析器**：序列输入、互补链生成、转录/翻译动画
3. **密码子表查询**：氨基酸-密码子映射、频率统计可视化
4. **分子量计算器**：DNA/RNA/蛋白质分子量计算、浓度换算
5. **tRNA 可视化器**：tRNA 三叶草结构动画展示

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|

## 3. 核心交互流程

### 3.1 DNA 分析流程

```
用户输入序列 → 实时验证(A/T/G/C) → 点击分析
                                      ↓
              ┌───────────────────────┼───────────────────────┐
              ↓                       ↓                       ↓
         显示互补链            转录为 mRNA            翻译为氨基酸链
         (逐字动画)           (链分离动画)           (核糖体动画)
```

### 3.2 密码子查询流程

```
用户输入/选择密码子 → 表格高亮对应行 → 弹出详情卡片(动画)
                                          ↓
                              显示氨基酸名称、简写、分子量、频率图
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深空生物实验室 - 深蓝色背景搭配青色/绿色荧光效果，营造分子世界的沉浸感
- **主色调**：
  - 背景：`#0a0e1a` (深空蓝)
  - 主色：`#00d4aa` (生物荧光青)
  - 强调色：`#7c3aed` (基因紫)
  - 碱基颜色：A-#ef4444(红), T-#3b82f6(蓝), G-#22c55e(绿), U-#f59e0b(橙)
- **字体**：
  - 标题：`"Orbitron"` (科技感)
  - 正文：`"IBM Plex Sans"` (清晰易读)
  - 序列/代码：`"JetBrains Mono"` (等宽)
- **按钮风格**：圆角胶囊形，悬停时发光边框 + 微微放大
- **布局风格**：卡片式模块，网格布局，响应式适配

### 4.2 动画设计原则

- **入场动画**：元素从下方淡入上移，间隔 100ms 依次出现
- **数据动画**：数字滚动、进度条填充、序列逐字显示
- **交互反馈**：按钮点击涟漪效果、卡片悬停上浮 + 阴影加深
- **背景动画**：DNA 双螺旋缓慢旋转，粒子漂浮

### 4.3 响应式设计

- **桌面端 (≥1024px)**：三列工具卡片，完整侧边导航
- **平板端 (768-1023px)**：两列卡片，精简导航
- **移动端 (<768px)**：单列卡片，底部导航栏，汉堡菜单

## 5. 技术约束

- 纯前端实现，无需后端服务器
- 支持现代浏览器（Chrome, Firefox, Safari, Edge 最新版）
- 首屏加载时间 < 3 秒
- 所有动画使用 CSS 或 Canvas 实现，确保 60fps 流畅度
