# tRna - 生物信息学工具集合

探索生命的遗传密码 🧬

一个专注于生物信息学的在线工具集合网站，帮助研究人员、学生和生物科技爱好者快速完成 DNA/RNA 序列分析、氨基酸查询等日常生物计算任务。

## 功能特点

- **DNA/RNA 分析器** - 序列输入、互补链生成、转录与翻译动画
- **密码子表** - 64 密码子与 20 氨基酸的可视化查询
- **分子量计算器** - DNA、RNA、蛋白质的分子量精确计算
- **tRNA 可视化** - 三叶草结构的动态展示与交互

## 技术栈

- React 18 + TypeScript
- Tailwind CSS
- Framer Motion (动画)
- Vite

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
